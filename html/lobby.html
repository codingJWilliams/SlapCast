What is this?

SnapCast is a hosted service to simplify the deployment of TV alerting systems. Using your private code you can instantly update all of the connected slaves with any number of cues

How much does it cost?

That's the best part.
It's absolutely free. SnapCast was written in Node.JS by Jay Williams to get experience with writing professional grade solutions.

What is a cue?

A cue is any type of content that can be displayed on a slave. Examples include:

  • Plain text
  • Images
  • Videos

Master Mode

You will recieve a secret code and a public code. The secret code will allow you to control slaves connected via the public code.

Begin a session

Slave Mode

Use a public code to slave a display to a specific master. This display will then take cues from the master.
